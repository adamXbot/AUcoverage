About this tool & data sources

Why this exists: from 30 June 2026, Australia’s Mobile Network Coverage Maps Industry Standard 2026 (ACMA) requires carriers to publish standardised 4G/5G maps labelled good / moderate / basic / no coverage, and the ACCC has warned providers that coverage claims must match. This open-source tool helps you sanity-check and compare, then sends you to each carrier’s official map for the authoritative answer.
